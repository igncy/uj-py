# mandelbrot.py

Mandelbrot set fractal simulation.

## Dependencies

- pygame

Written in Python 3.13 with pygame 2.6.1.

## Usage

```python
from mandelbrot import App
app = App()
app.run()
```

## Controls

- WASD / Arrow keys — move the camera
- E, Page Up — zoom in
- <kbd>&#8592;</kbd>
    left arrow: ← &#8592;
    upward arrow: ↑ &#8593;
    right arrow: → &#8594;
    downward arrow: ↓ &#8595;
Q, Page Down — zoom out
Escape — quit

|                  |                 |
|:----------------:|:----------------|
| WASD, Arrow keys | move the camera |
